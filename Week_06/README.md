
本周主要学习动态规划算法。动态规划和递归或分治没有根本上的区别，都是找到重复子问题，不同在于动态规划是看有无最优的子结构，其中途可以淘汰次优解。

在用动态规划解决问题时，我们可以把问题分解为多个阶段，每个阶段对应一个决策。我们记录每一个阶段可达的状态集合(去掉重复的)，然后通过当前阶段的状态集合，动态地往前推进。具体可分为三步： 1）化繁为简成为各类子问题； 2）定义好状态空间； 3）动态规划的方程。
